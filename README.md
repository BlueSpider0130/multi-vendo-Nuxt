# Providr

Providr is a Nuxt project for multi-vendor food ordering platform

## Installation

Use the package yarn to install Providr.

```bash
yarn install
```

## To start in development mode


```bash
yarn dev
```


## To start in production mode


```bash
yarn run generate
```
```bash
yarn run start
```
